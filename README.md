## Information

This plugin adds neovim support for `.reamake` template files.  
For more information on how the language is used, visit the [main repo](https://github.com/simon-danielsson/reamake).  
  
### Features
  
- Filetype detection  
- Syntax highlighting  
- Indentation  
- Ftplugin defaults  
  
---
<div id="install"></div>
  
## Install
  
``` lua
vim.pack.add({
    {
        src = "https://github.com/simon-danielsson/reamake.nvim",
    },
})
```
